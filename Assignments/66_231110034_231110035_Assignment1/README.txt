To run the file, for simplicity, please keep the .py files and the "Data" folder in the same directory.
Else, please edit the path in reader.GetFileName() accordingly.

Q1. Just run the file, it will ask for a pressure value as input.
Enter the value for which you want the isocontour.
We will get a vtp file as output in the following format
e.g. "grp66_test_val470.vtp" for pressure value of 470.


Q2. Also, by default, Phong shading will be off.
Therefore, to turn it on, it will ask of input. 
If input is given as "Yes" then Phong shading is turned on.
